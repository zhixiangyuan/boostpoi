# BoostPOI

对 poi 的二次封装，将表格坐标封装成相对坐标，方便模块化 Excel。

# 安装

```xml
<dependency>
    <groupId>com.github.zhixiangyuan</groupId>
    <artifactId>boostpoi</artifactId>
    <version>LASTEST</version>
</dependency>
```
